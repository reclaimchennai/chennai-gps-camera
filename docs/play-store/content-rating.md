# Content rating (IARC questionnaire) — answers

Play Console → App content → **Content ratings**. Answer the IARC questionnaire
with these; the expected result is **Everyone / PEGI 3 / rated for all ages**.

- **App category:** Utility / Productivity / Communication (not a game).
- **Violence:** None.
- **Sexuality / nudity:** None.
- **Profanity / crude humour:** None.
- **Controlled substances (drugs/alcohol/tobacco):** None.
- **Gambling / simulated gambling:** None.
- **Real-money gambling:** No.
- **User-generated content shared with others?** The app has **no social feed
  or public sharing surface**. Users can send a private bug report to the
  developer (text + optional screenshot) — declare this if asked, but there is
  no user-to-user or public UGC.
- **User-to-user communication / chat:** None.
- **Shares user's physical location with other users?** **No.** Location is
  stamped onto the user's own photos on their device; it is not shared to other
  users through the app.
- **Digital purchases:** None.
- **Ads:** None.

**Contact email for the rating certificate:** contact@reclaimchennai.city
